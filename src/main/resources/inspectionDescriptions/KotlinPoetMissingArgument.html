Reports a KotlinPoet format string that contains a placeholder (%L, %S, %N, %T, %M, or %P) with no corresponding argument supplied to the call.

This mismatch causes an IllegalArgumentException at runtime when KotlinPoet processes the format string.

Example (flagged):


CodeBlock.builder().add("val %N = %S")
//                            ^^   ^^ no arguments supplied → runtime exception

Correct usage:


CodeBlock.builder().add("val %N = %S", "myVar", "hello")

Because the correct argument value cannot be inferred, no automatic quick-fix is available. Add the missing argument manually.
