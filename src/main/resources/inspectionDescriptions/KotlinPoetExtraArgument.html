Reports an argument passed to a KotlinPoet call that has no matching placeholder in the format string.

For positional and relative calls, a surplus argument causes an IllegalArgumentException at runtime. For named calls the severity is informational because the same map may be intentionally shared across multiple format strings.

Example – relative (flagged):


CodeBlock.builder().add("Hello %L", name, extraValue)
//                                        ^^^^^^^^^^ no placeholder for this argument

Example – named (informational):


CodeBlock.builder().addNamed("%food:L", mapOf("food" to "tacos", "count" to 3))
//                                                               ^^^^^^^^^^^^^ unused key

Quick fix: Remove extra argument – deletes the surplus argument from the call site.
