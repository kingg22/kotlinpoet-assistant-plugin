Reports an argument whose resolved type is incompatible with the placeholder kind it is bound to in a KotlinPoet format string.

PlaceholderExpected typeSeverity
%S / %PString / CharSequence / CodeBlockInformation
%TTypeName, KClass, Class, Type, TypeMirror, ElementWarning
%MMemberNameWarning
%NCharSequence, ParameterSpec, PropertySpec, FunSpec, TypeSpec, ContextParameterWarning

Example (flagged):


val block = CodeBlock.builder().add("%T", "not a type")
//                                       ^^^^^^^^^^^^ String passed where TypeName expected

Correct usage:


val block = CodeBlock.builder().add("%T", String::class)

No automatic fix is offered because the correct replacement type cannot be inferred.
