# KotlinPoet Assistant IntelliJ Plugin

<!-- Plugin description -->
**KotlinPoet Assistant** provides lightweight IDE assistance when writing KotlinPoet code.
It validates `CodeBlock` format strings at edit time, highlights placeholders, checks argument count and types,
and helps prevent common runtime errors, all in a non-intrusive way, and fully compatible with Kotlin K2.

Future versions will expand support to more KotlinPoet APIs and offer code previews. Stay tuned!
